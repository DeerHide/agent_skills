# Epic and User Story Templates

Copy-paste and fill in the placeholders. Match the structure in the jira-epics-stories skill.

## Epic Template

```markdown
- **Title:** [Short epic title]
- **Goal:** [One sentence goal]
- **Scope:** [What is in scope / out of scope]
- **Success outcomes:**
  - [Measurable outcome 1]
  - [Measurable outcome 2]
- **Stories:** [List of story titles or keys once defined]
```

## User Story Template

```markdown
- **Title:** [Short story title]
- **As a** [role], **I want** [action/capability] **so that** [benefit].
- **Acceptance criteria:**
  - Given [context], when [action], then [observable result].
  - [Additional criterion in Given/When/Then or checklist form.]
- **Story points:** [Optional, if requested]
```
